# Roadmap

1. **Event protocol** - extend payload schema with role/location and richer step metadata.
2. **FastAPI broker** - hardening, optional auth, and metrics.
3. **Office frontend** - richer zones, visuals, and optional PixiJS migration if needed.
4. **Reference integrations** - examples for CrewAI, LangGraph, and custom emitters.
5. **`client/` helper** - minimal Python helper to emit `/event` payloads from any runtime.
